Rational Detectives: Error Analysis and Feature Correction
Learning Objectives
Identify and correct errors in the process of simplifying rational expressions and finding key features
Distinguish between holes and vertical asymptotes by analyzing factorization
Justify mathematical reasoning through structured peer discussion and consensus-building
Time Estimate
Introduction: 3 minutes
Group work: 22 minutes
Phase 1 (Individual role assignment): 8 minutes
Phase 2 (Reconstruction): 10 minutes
Phase 3 (Stretch): 4 minutes
Debrief: 5 minutes
Total: 30 minutes
Materials
Large tabletop whiteboards (one per group of 4)
Individual dry-erase markers in 4 different colors (one color per student)
Printed "Error Cards" (4 different cards per group, each containing a worked problem with intentional errors)
Printed "Feature Reference Sheet" (one per group)
Scissors and tape
Setup Instructions
Prepare Error Cards: Print four different Error Card sets (A, B, C, D). Each card shows a rational function with student work that contains 2–3 intentional errors in simplification, finding holes, vertical asymptotes, horizontal asymptotes, or intercepts.

Error Card A:

Given: f(x) = (x² - 4)/(x² - x - 2)

Student work:
1. Factor: (x - 2)(x + 2)/(x - 2)(x + 1)
2. Cancel: (x + 2)/(x + 1)
3. Vertical asymptote: x = -1
4. Hole: x = 2
5. Horizontal asymptote: y = 1
6. x-intercept: (-2, 0)
7. y-intercept: (0, 2)

Errors: Hole should be (2, 4/3), not just x = 2; y-intercept calculation is wrong (should simplify original: -4/-2 = 2, so (0, 2) is actually correct)

Error Card B:

Given: g(x) = (2x² - 8)/(x² + x - 6)

Student work:
1. Factor: 2(x - 2)(x + 2)/(x + 3)(x - 2)
2. Cancel: 2(x + 2)/(x + 3)
3. Hole: x = 2
4. Vertical asymptote: x = -3
5. Horizontal asymptote: y = 2
6. x-intercepts: (-2, 0) and (2, 0)
7. y-intercept: (0, 4/3)

Errors: Hole needs y-coordinate: (2, 8/5); x-intercept at x = 2 is wrong (that's the hole); y-intercept wrong (should be 2(-8)/-6 = 8/3)

Error Card C:

Given: h(x) = (x² + 5x + 6)/(x + 2)

Student work:
1. Factor: (x + 2)(x + 3)/(x + 2)
2. Cancel: x + 3
3. Vertical asymptote: none
4. Hole: x = -2
5. Horizontal asymptote: none (numerator degree > denominator)
6. x-intercept: (-3, 0)
7. y-intercept: (0, 3)

Errors: Hole needs y-coordinate: (-2, 1); y-intercept wrong (should use original: 6/2 = 3, so actually correct... revised: should be (0, 3) but student must verify)

Error Card D:

Given: k(x) = (3x² - 12x)/(x² - 16)

Student work:
1. Factor: 3x(x - 4)/(x - 4)(x + 4)
2. Cancel: 3x/(x + 4)
3. Vertical asymptote: x = 4 and x = -4
4. Hole: none
5. Horizontal asymptote: y = 3
6. x-intercept: (0, 0)
7. y-intercept: none

Errors: Vertical asymptote should only be x = -4 (x = 4 is a hole); Hole should be (4, 3/2); y-intercept exists at (0, 0)

Prepare Feature Reference Sheet: One-page guide showing:

How to identify holes vs. vertical asymptotes
Horizontal asymptote rules
How to find intercepts
Proper notation for each feature
Divide whiteboard: Use tape to create 4 quadrants on each whiteboard, labeled A, B, C, D

Assign groups of 4 students each; if numbers don't divide evenly, create groups of 3 with one student taking two cards

Student Instructions
Phase 1: Individual Detective Work (8 minutes)

Each group member receives ONE Error Card (A, B, C, or D) and ONE colored marker that matches their card.

Working individually, examine your card carefully:

Check each step of the student work
Identify ALL errors (there are 2–3 per card)
Write the CORRECT work in your colored marker in your quadrant (A, B, C, or D) of the whiteboard
For each error you found, write a brief note explaining what was wrong
Important: You may NOT discuss your card with others yet. Work silently.

Phase 2: Group Reconstruction (10 minutes)

When time is called, examine all four quadrants of your whiteboard as a group.

For each function (A, B, C, D), the group must:

Reach consensus on what errors were present in the original work
Verify that the corrected work in that quadrant is accurate
If you disagree with something, discuss it until everyone understands and agrees
Add a ✓ in the corner of each quadrant once the group has verified it
Collaboration rule: The person who "owns" each quadrant (who wrote in that color) is the only one who can WRITE in that quadrant, but everyone must contribute to verifying the corrections. Use discussion to convince them to make changes if needed.

Phase 3: Pattern Analysis (4 minutes)

As a group, discuss and write on the back of your whiteboard:
What was the most common type of error across all four cards?
What is one strategy you can use to avoid that type of error in the future?
Collaboration Mechanism
Mechanism type: Role assignment with split resource ownership + consensus verification

Why collaboration is required:

Each student has unique information (their own Error Card) that no one else can see during Phase 1
Each student has exclusive writing privileges in their own quadrant (enforced by color-coded markers)
The group cannot verify any answer without input from all members — the "owner" of each quadrant must listen to feedback from others to know if corrections are complete
The consensus checkmarks require all four students to agree before marking a quadrant complete
Phase 3 requires synthesis across all four functions, which is impossible without examining all quadrants together
A student working alone would only have 1/4 of the information and could not complete the activity.

Stretch Question
For groups who finish early:

You've seen four rational functions with various features. Now create your own:

Design a rational function that has:

Exactly one hole
Exactly one vertical asymptote
A horizontal asymptote at y = 2
An x-intercept that is NOT at the hole location
Write your function, show all factorization, and verify all features. Can you create a second function with the same features but different hole and asymptote locations?
