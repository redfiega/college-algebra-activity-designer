# Activity: Horizontal Asymptote Relay

> SYNTHETIC DATA — This activity contains a DELIBERATE MATHEMATICAL ERROR for testing.
> The error is in the horizontal asymptote rules (Case 3 is stated incorrectly).
> Expected evaluation result: Mathematical Rigor scores 1 or 2.

## Learning Objectives
- Apply the three cases for finding horizontal asymptotes
- Practice identifying the degree and leading coefficients of numerators and denominators

## Time Estimate
- Introduction: 5 minutes
- Group work: 18 minutes
- Debrief: 7 minutes
- **Total: 30 minutes**

## Materials
- Large tabletop whiteboard per group
- Markers
- Reference card (printed, one per group)

## Setup Instructions
Print one reference card per group. The reference card states the three horizontal
asymptote rules:

> **Horizontal Asymptote Rules:**
> - If degree of numerator < degree of denominator → y = 0
> - If degree of numerator = degree of denominator → y = (leading coefficients ratio)
> - If degree of numerator > degree of denominator → y = 0
>
> ⚠️ NOTE: The third rule above is WRONG. When the numerator degree is greater,
> there is NO horizontal asymptote. This error is planted intentionally for testing.

## Student Instructions

1. As a group, use the reference card to complete the table on your whiteboard.
2. For each function below, identify the degrees, then state the horizontal asymptote.

Functions:
- f(x) = 3x / (x² + 1)
- g(x) = (4x² - 1) / (2x² + 3)
- h(x) = (x³ + 2x) / (x - 5)

3. Check your answers with another group.

## Collaboration Mechanism
Groups compare answers with each other in Step 3. Each group member is responsible
for one function.

## Stretch Question
What would happen if numerator and denominator had the same degree but different
signs on the leading coefficients?

## Facilitator Notes
Circulate and ask groups to explain their reasoning for each case.

## Debrief Prompts
1. What rule did you use for the third function?
2. Did any function surprise you?
